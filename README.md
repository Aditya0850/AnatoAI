# AnatoAI – 3D Interactive Human Health Assistant

AnatoAI is an interactive 3D web application that allows users to explore the human body and analyze health symptoms using AI.

Users can click on different parts of a 3D human body model, describe their symptoms, and receive AI-generated insights about possible causes, remedies, and medications.

⚠️ This project is for educational purposes and does not replace professional medical advice.

---

## Features

- Interactive **3D Human Body**
- Hover and click body parts
- Symptom selection system
- AI-powered health analysis
- Remedies and medication suggestions
- Responsive modern UI

---

## Tech Stack

Frontend:
- React.js
- Three.js
- React Three Fiber
- TailwindCSS

Backend:
- Node.js
- Express.js

AI:
- OpenAI API

3D Model:
- GLTF / GLB Human Body Model

---

## How It Works

1. User opens the website
2. A 3D human body appears
3. User hovers over body parts
4. Clicks the part with pain
5. A symptom popup appears
6. User describes symptoms
7. AI analyzes the symptoms
8. System returns:
   - Possible causes
   - Remedies
   - Medication suggestions
   - Doctor recommendation
